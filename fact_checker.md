# Chrome Gemini 扩展：事实校验技能

在Chrome浏览器的Gemini对话框中添加自定义技能，实现一键事实核验。

## 技能配置

### 基本信息

- **触发命令**：`/事实校验`
- **名称**：事实校验
- **说明**：先分段总结当前内容的概要，然后搜索、分析、验证内容的准确性。

### Prompt模板

```
先分段总结当前内容的概要，然后搜索、分析、验证内容的准确性。

格式采用表格形式：
- 第一列"总结"：分段内容的核心观点（一句话）
- 第二列"事实"：搜索最新相关新闻、权威来源，核实后给出结论
- 第三列"验证"：✅（正确）、❌（错误）、⚠️（部分正确/有争议）、❓（无法验证）

对于论点，标注是否为业界主流观点，并注明争议点。
对于无法验证的内容，说明原因（如：需要专业实验、数据不足等）。

最后给出：
1. 整体准确性评估
2. 核心正确结论总结
3. 主要信息来源

使用简体中文，尽量简短。
```

## 配置步骤

### 步骤1: 打开Gemini扩展设置

1. 在Chrome浏览器中点击右上角的 **Gemini图标**
2. 点击对话框中的 **设置图标** ⚙️
3. 找到 **"技能"** 或 **"Skills"** 选项

### 步骤2: 添加新技能

1. 点击 **"添加技能"** 或 **"+"**
2. 填写以下信息：

**名称**：
```
事实校验
```

**触发命令**：
```
/事实校验
```

**说明**：
```
先分段总结当前内容的概要，然后搜索、分析、验证内容的准确性。

格式采用表格形式：
- 第一列"总结"：分段内容的核心观点（一句话）
- 第二列"事实"：搜索最新相关新闻、权威来源，核实后给出结论
- 第三列"验证"：✅（正确）、❌（错误）、⚠️（部分正确/有争议）、❓（无法验证）

对于论点，标注是否为业界主流观点，并注明争议点。
对于无法验证的内容，说明原因（如：需要专业实验、数据不足等）。

最后给出：
1. 整体准确性评估
2. 核心正确结论总结
3. 主要信息来源

使用简体中文，尽量简短。
```

3. 点击 **"保存"**

## 使用方法

### 方法1: 直接输入命令

1. 打开要核验的网页
2. 点击Chrome右上角的 **Gemini图标**
3. 在对话框中输入：
   ```
   /事实校验
   ```
4. 按Enter发送
5. Gemini会自动读取当前页面内容并执行核验

### 方法2: 选中内容后核验

1. 在网页中选中要核验的文本
2. 右键 → **使用Gemini分析**（或类似选项）
3. 在Gemini对话框中输入：
   ```
   /事实校验
   ```
4. Gemini会核验选中的内容

## 高级配置

### 添加更多技能

你可以创建多个技能用于不同场景：

#### 技能1: 快速核验
```
/快速核验
```
Prompt：
```
快速判断以下内容的真实性，给出：
1. 主要论点（1-2句）
2. 是否正确（✅/❌/⚠️）
3. 简要理由

使用简体中文。
```

#### 技能2: 深度分析
```
/深度分析
```
Prompt：
```
对以下内容进行深度事实核查：

1. 提取所有可验证的事实陈述
2. 逐条搜索权威来源验证
3. 分析论证逻辑是否严密
4. 识别可能的偏见或误导
5. 给出综合评估

格式要求：
- 每个事实单独列出
- 标注信息来源和可信度
- 最后给出整体评分（1-10分）

使用简体中文。
```

#### 技能3: 新闻核验
```
/新闻核验
```
Prompt：
```
核验此新闻报道的真实性：

1. 检查新闻来源是否权威
2. 验证核心事实是否准确
3. 查找是否有其他媒体报道
4. 识别可能的偏见或立场
5. 检查是否有后续更正或撤回

输出格式：
- 来源可信度：⭐⭐⭐⭐⭐（5星制）
- 事实准确性：✅/❌/⚠️
- 媒体覆盖：有多少家媒体报道
- 潜在偏见：无/轻微/明显
- 综合评估：可信/存疑/不可信

使用简体中文。
```

## 注意事项

### 1. Gemini的搜索能力

- Gemini会自动搜索网络验证信息
- 对于时效性强的新闻，建议查看发布时间
- 对于专业领域，建议交叉验证多个来源

### 2. 验证符号说明

- ✅ **正确**：有明确证据支持
- ❌ **错误**：有明确证据反驳
- ⚠️ **部分正确/有争议**：部分准确或有不同观点
- ❓ **无法验证**：缺乏足够信息或需要专业验证

### 3. 局限性

- Gemini可能无法访问付费墙后的内容
- 对于最新发生的事件，可能有延迟
- 对于高度专业的领域，建议咨询专家

## 技巧提示

### 1. 快速触发

电脑端记住命令 `/事实校验`，可以快速输入而不需要每次都输入完整prompt。iPhone/iPad端分享给“事实核验”即可。

### 2. 上下文核验

电脑端如果要对页面特定部分核验，先选中文字，再输入 `/事实校验`。

### 3. 追问

核验后可以继续追问：
- "这个结论有争议吗？"
- "还有其他观点吗？"
- "给我看原始来源"

### 4. 保存常用核验

对于经常核验的主题（如健康、科技新闻），可以创建专门的技能变体。

## 常见问题

### Q1: Gemini无法访问当前页面

**原因**：某些网站有反爬机制或需要登录

**解决**：
- 复制页面内容到Gemini对话框
- 或使用iOS Shortcuts的分享功能

### Q2: 核验结果不准确

**原因**：Gemini的搜索结果可能有偏差

**解决**：
- 查看Gemini提供的信息来源
- 交叉验证多个来源
- 对于重要信息，人工核实

### Q3: 技能命令不生效

**原因**：可能是命令格式错误

**解决**：
- 确认命令格式：`/事实校验`（注意是正斜杠）
- 检查技能是否正确保存
- 尝试重新添加技能
